# States & Territories, 1861 — Map Quiz Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "States & Territories, 1861" map with Learn mode (click-to-explore with tooltips) and Quiz mode (click-to-identify) to the westward-expansion unit's map quiz, plus mobile-friendly fallbacks.

**Architecture:** Region data extracted to a separate `map-1861-data.js` file exposing `window.MAP_1861_REGIONS` and `window.MAP_1861_DRAW_ORDER`. Each region includes SVG paths, historical data (year, status, capital), and Union/Confederate/Border/Territory allegiance. New methods `_start1861Learn()` and `_start1861Quiz()` handle the two modes. Quiz supports subset filters (All, States Only, Territories Only, Union, Confederate, Border). A new mode card in the existing `_showModeSelector()` launches Learn mode. Mobile detection switches Quiz mode to multiple-choice buttons. All new CSS classes prefixed `mq-1861-` or `mq-learn-`.

**Tech Stack:** Vanilla JS, CSS custom properties, SVG, DOM APIs (no innerHTML)

**Spec:** `docs/plans/2026-03-30-civil-war-map-design.md`
